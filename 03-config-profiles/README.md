# Configuration Profiles

## Overview
This section contains Windows device configuration profiles deployed via Microsoft Intune.
These profiles form the baseline configuration required for security, compliance, and stable device operation.

The configuration profiles are applied to Windows 10/11 devices enrolled via Autopilot.

---

## Included profiles

### Device Baseline
Base configuration profile used to disable consumer features and apply common enterprise settings.

### Device Restrictions
Restrictions applied to Windows devices to limit unwanted functionality and enforce corporate standards.

### BitLocker
Disk encryption configuration to protect data at rest and escrow recovery keys to Microsoft Entra ID.

### Windows Update
Update ring configuration to control update behavior, active hours, and restart experience.
